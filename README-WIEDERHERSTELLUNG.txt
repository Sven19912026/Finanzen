FINANZAPP - CLOUD-BACKUP-RETTUNG

1. Die Dateien index.html, cloud-sync.js und sw.js auf GitHub ersetzen.
2. GitHub Pages Deployment abwarten.
3. App öffnen und anmelden.
4. Oben auf "Cloud-Backups" klicken.
5. Das Backup vom 21.07.2026 auswählen.
6. "Wiederherstellen" bestätigen.

Die App sichert den aktuellen Cloud-Stand vor der Wiederherstellung zusätzlich.
Ein leerer lokaler Stand darf künftig keine vorhandenen Cloud-Schulden oder
Monatszahlungen mehr automatisch überschreiben.
